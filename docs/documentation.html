📱 Introduction

ZeroRange est un système d'entraînement interactif conçu pour maîtriser les fonctionnalités du Flipper Zero. Il propose une série de défis pratiques dans différents domaines : iButton, NFC, RFID et infrarouge.

🎯 Objectif : Accumuler des points en complétant des défis de difficulté croissante. Score maximum : 40 points

Matériel requis

  • Raspberry Pi (testé sur Pi 5)
  • Adafruit RGB LCD Shield 16x2 avec boutons
  • Flipper Zero
  • Interface iButton (GPIO 4)

🧭 Navigation générale

Les boutons

Le système utilise 5 boutons pour la navigation :

Bouton Fonction Description
▲ UP Naviguer vers le haut Permet de remonter dans une liste d'options ou de menu
▼ DOWN Naviguer vers le bas Permet de descendre dans une liste d'options ou de menu
● SELECT Sélectionner / Confirmer Valide le choix actuel ou entre dans un sous-menu
◄ LEFT Retour Revient au menu précédent ou annule une action
► RIGHT Réservé Fonction contextuelle selon le menu

Comprendre l'affichage LCD

L'écran LCD 16x2 affiche les informations sur deux lignes de 16 caractères :

Format d'affichage standard

>iButton 0/40 U/D SEL=Go L=Bk
  • > : Indicateur de sélection (pointe vers l'élément actif)
  • iButton : Nom du module ou de l'option sélectionnée
  • 0/40 : Score actuel / Score maximum
  • U/D : Up/Down - Utiliser les boutons haut/bas
  • SEL=Go : SELECT pour valider
  • L=Bk : LEFT pour retour (Back)

Principes de navigation

  1. Navigation cyclique : Appuyer sur DOWN depuis le dernier élément revient au premier (et vice-versa)
  2. Validation immédiate : SELECT entre directement dans le menu ou lance l'action
  3. Retour rapide : LEFT permet de revenir en arrière à tout moment
  4. Indicateur visuel : Le symbole > indique toujours l'élément actif
💡 Astuce : Les messages d'aide sur la ligne 2 sont contextuels et s'adaptent au menu actuel pour vous guider.

Le menu principal est le point d'entrée du système. Il affiche tous les modules disponibles et votre score global.

Structure du menu

>iButton 0/40 U/D SEL=Go L=Bk

  2. Le symbole > indique l'option sélectionnée

🔑 Module iButton

Le module iButton propose 3 défis de difficulté progressive pour maîtriser la lecture, le clonage et l'émulation de clés iButton (Dallas 1-Wire).

Menu iButton

iBtn Challenges 0/3 SEL=Go L=Bk

L'affichage montre le nombre de défis complétés (0/3). Appuyez sur ● SELECT pour lancer un défi.

Défi 1 : Touch & Read (10 points)

Objectif

Apprendre à lire une clé iButton avec le Flipper Zero

Déroulement

Challenge 1 Touch & Read
  1. Le système affiche le défi pendant 3 secondes
  2. L'écran passe en mode attente :
    Waiting for key. Touch to reader.
  3. Touchez la clé iButton au lecteur connecté au Raspberry Pi
  4. Le système détecte la clé et affiche son ID (ex: 01 23 45 67 89 AB CD EF)
  5. L'écran affiche les instructions Flipper :
    Flipper: iBtn Read -> Save
  6. Sur votre Flipper Zero :
    • Allez dans iButton
    • Sélectionnez Read
    • Touchez la clé au lecteur du Flipper
    • Appuyez sur Save
  7. Retouchez la clé au lecteur du Raspberry Pi pour valider

Validation

✅ Succès :
SUCCESS! +10pts Challenge Done!
⏱️ Timeout : Si aucune clé n'est détectée pendant 30 secondes, le défi échoue et vous revenez au menu.

Défi 2 : Clone Master (10 points)

Objectif

Apprendre à cloner une clé iButton sur une clé vierge

Déroulement

Challenge 2 Clone Master
  1. Le système affiche le défi pendant 3 secondes
  2. Touchez la clé source au lecteur
  3. Le système affiche l'ID de la clé à cloner
  4. Instructions pour le Flipper :
    Flipper: Read then Write
  5. Sur votre Flipper Zero :
    • Read la clé source
    • Save les données
    • Allez dans Saved
    • Sélectionnez la clé sauvegardée
    • Choisissez Write
    • Touchez une clé vierge pour écrire
  6. Touchez la clé clonée au lecteur du Raspberry Pi
  7. Le système vérifie que l'ID correspond

Validation

✅ Succès : Le clone correspond à l'original (+10 points)
❌ Échec : Si l'ID ne correspond pas, le défi échoue (0 points)

Défi 3 : Emulate Pro (10 points)

Objectif

Maîtriser l'émulation d'une clé iButton avec le Flipper Zero

Déroulement

Challenge 3 Emulate Pro
  1. Le système affiche le défi pendant 3 secondes
  2. Touchez la clé au lecteur pour obtenir l'ID cible
  3. Instructions d'émulation :
    Flipper: Saved key -> Emulate
  4. Sur votre Flipper Zero :
    • Allez dans iButton
    • Sélectionnez Saved
    • Choisissez la clé à émuler
    • Appuyez sur Emulate
  5. Maintenez le Flipper près du lecteur (mode émulation actif)
  6. Le système détecte l'émulation et vérifie l'ID

Validation

✅ Succès : L'émulation est détectée et l'ID correspond (+10 points)
💡 Astuce : L'émulation doit rester active pendant toute la durée de la lecture. Gardez le Flipper en position jusqu'à la confirmation de succès.

⚙️ Paramètres

Le menu Settings permet de consulter vos statistiques et de réinitialiser votre progression.

Menu Settings

Settings 1:Stat 2:Reset

Option 1 : Statistiques

Appuyez sur ● SELECT pour voir vos statistiques détaillées.

Score: 20/40 2/3 Tries:5

Détails de l'affichage

  • Score: 20/40 : Score actuel sur le maximum possible
  • 2/3 : Nombre de défis complétés sur le total
  • Tries:5 : Nombre total de tentatives effectuées

L'écran s'affiche pendant 5 secondes, puis revient automatiquement au menu Settings.

Option 2 : Réinitialisation

Permet de réinitialiser tous les scores et l'historique des défis.

⚠️ Attention : Cette action est irréversible !

Processus de réinitialisation

  1. Depuis le menu Settings, sélectionnez l'option Reset
  2. Un écran de confirmation apparaît :
    Reset scores? 1:Yes 5:No
  3. Appuyez sur ● SELECT (bouton 1) pour confirmer
  4. Appuyez sur ◄ LEFT (bouton 5) pour annuler

Confirmation de réinitialisation

Scores Reset! Progress cleared

Après réinitialisation, tous les défis redeviennent disponibles et le score retourne à 0/40.

Sortir des paramètres

Appuyez sur ◄ LEFT depuis le menu Settings pour revenir au menu principal.
